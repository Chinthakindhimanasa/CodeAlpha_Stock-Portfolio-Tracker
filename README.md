# Stock Portfolio Tracker

## Description
This project calculates total stock investment value based on stock quantity entered by the user.

Stock prices are stored using a dictionary.

## Features
- User input for stock name and quantity
- Calculates investment value
- Displays total investment
- Saves results into a text file

## Technologies Used
- Python
- Dictionary
- File Handling

## How to Run
1. Open terminal
2. Run command:

python stock_tracker.py

## Sample Output
Enter stock name: AAPL
Enter quantity: 5

Investment Value: 900

## Concepts Used
- dictionary
- arithmetic operations
- file handling
- loops
